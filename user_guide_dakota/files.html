Работа с файлами

  • Файлы хранятся в папке /uploads/files/.
  • Вы можете использовать их для вставки картинок и ссылок на загрузки в текст публикаций, страницах загрузки и т.п.
  • Файлы разбиваются на группы Documents, Images, Archives, Music, Video по расширению
  • Запрещены файлы php,php3,php4,php5,phtml,phps,exe,bin,bat,sh
  • Максимальный размер файла указан в строке upload_max_filesize файла php.ini.
  • Максимальный размеры для картинок 2560 на 1600px
  • Пусть к php.ini указан в строке Loaded Configuration File phpinfo. Обратитесь к администратору, если у Вас нет туда доступа.
  • Нажмите кнопку Copy link, чтобы использовать ссылку на файл в тексте Публикации
  • Поведение кнопки Copy link зависит от настроек браузера. Если она не работает, копируйте ссылку, через меню правой кнопки на Copy link
  • Файлы НЕ выводятся на сайте сами
