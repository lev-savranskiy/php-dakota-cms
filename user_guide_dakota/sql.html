Работа с Базой данных

Информация

  • Для работа с Базой данных используется программа SQL Buddy.
  • Функциональность не гарантируется в Internet Explorer.
  • Функциональность проверялась в конфигурации с ОДНИМ сервером БД. Если Ваш хостинг-провайдер предоставляет выбор сервера БД для работы, то возможно без дополнительной доработки SQL Buddy не запустится. В таком случае используйте средства, предоставленные провайдером.
  • В целях безопасности вы можете удалить папку sqlbuddy из проекта.
