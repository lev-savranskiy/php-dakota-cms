Информация

  • В Dakota CMS используются 4 вида страниц ошибок: 404, db, php, geleral. Они отрабатываются автоматически.
  • Пример ошибки 404
  • Внешний вид можно поменять с помощью стиля /templates/common/css/error.css
  • Dakota CMS поставляется с двумя языками (рус, англ). Для отображения ошибок на других языках, обратитесь к разделу i18n
